# 児童関係手当制度 法改正・国会審議一次資料JSON化プロジェクト

このディレクトリは、児童手当法、児童扶養手当法、特別児童扶養手当等の支給に関する法律、および関連施行令について、日本法令索引と国会会議録を一次資料として、法改正履歴・趣旨説明・質疑を構造化JSONにするための作業用ディレクトリです。

Codexは `instructions/codex_task.md` に従って作業してください。


## Codex実行環境での取得制約と手動入力方式

Codex実行環境から `hourei.ndl.go.jp` へ直接アクセスすると、`Tunnel connection failed: 403 Forbidden` となる場合があります。
このため、Web取得はCodexの責務から外し、ユーザーがブラウザで保存した日本法令索引ページ（HTML/TXT）を入力として扱います。

- 手動保存先: `project/data/manual_sources/hourei/`
- Codexの処理: 保存済みファイルを解析し、法令沿革・改正法リンク・審議経過リンクの抽出下書きを作成
- 不明値は `null` を維持し、推測で補完しない
